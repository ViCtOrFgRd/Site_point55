# 🎨 GUIA RÁPIDO - Adicionar Cores com Gradiente

## ⚡ Uso Rápido em 3 Passos

### 1️⃣ Abrir o Painel Admin
```
1. Ir para: Painel Admin → Produtos
2. Clicar no produto ou criar novo
3. Scrollar até seção "Variações"
```

### 2️⃣ Adicionar Cor Simples (ex: Azul)
```
Campo: "Cores Disponíveis"
Digite: Azul
Clique: Adicionar
✅ Resultado: Bolinha AZUL SÓLIDA
```

### 3️⃣ Adicionar Variação Gradiente (ex: Branco/Azul)
```
Campo: "Cores Disponíveis"
Digite: Branco/Azul
Clique: Adicionar
✅ Resultado: Bolinha com GRADIENTE (meio branco, meio azul)
```

## 💡 Dicas Importantes

1. **Use "/" sem espaços**
   - ✅ `Branco/Azul`
   - ❌ `Branco / Azul`

2. **Copie o nome EXATAMENTE**
   - ✅ `Azul`
   - ❌ `azul` ou `AZUL`

3. **Máximo 5-7 cores por produto** (recomendado)
   - Deixa mais limpo
   - Cliente não fica confuso

4. **Ordem das cores importa pouco**
   - `Branco/Azul` e `Azul/Branco` funcionam igual
   - Apenas primeira cor fica como "dominante"

5. **Tamanhos funcionam igual**
   - P, M, G, GG (ou customize)
   - Sem /, sem gradiente

## ❓ Dúvidas?

### P: A cor não aparece?
R: Verifique se digitou EXATAMENTE como está na lista. Sem espaços extras!

### P: O gradiente não aparece diagonal?
R: É normal! Gradiente é intencional diagonal (135°) para melhor visual.

### P: Posso combinar 3+ cores?
R: Agora só 2. Versão futura pode suportar mais.

### P: Posso usar cor customizada?
R: Use uma das 28 cores da lista. Se precisar outra, solicite ao desenvolvedor.

### P: Como remover uma cor?
R: Clique no X ao lado da cor na lista de cores adicionadas.
