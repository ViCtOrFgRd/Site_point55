## 🚀 COMO USAR

### Adicionar Cor Simples
Admin Panel → Editar Produto → Variações → Cores Disponíveis
Digite: Azul
Resultado: Bolinha AZUL SÓLIDA ✅

### Adicionar Variação Gradiente
Admin Panel → Editar Produto → Variações → Cores Disponíveis
Digite: Branco/Azul
Resultado: Bolinha com GRADIENTE (meio branco, meio azul) ✅

## 🎯 PASSO-A-PASSO RÁPIDO

1. Abra admin/produtos/[id] (edição)
2. Scrolle até "Variações"
3. Campo "Cores Disponíveis"
4. Digite: Azul
5. Clique: Adicionar
6. Digite: Branco/Azul
7. Clique: Adicionar
8. Salve o produto
9. Veja no site - as cores aparecem com gradiente! ✅

═══════════════════════════════════════════════════════════════════════════════

## ⚡ DICAS IMPORTANTES

✅ CORRETO:
  Azul
  Branco/Azul
  Preto

❌ ERRADO:
  azul (use Azul)
  Branco / Azul (não use espaços)
  #0066CC (use o nome)
