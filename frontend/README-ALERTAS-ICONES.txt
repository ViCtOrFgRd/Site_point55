💻 PADRÃO DE CÓDIGO ESTABELECIDO
───────────────────────────────────────────────────────────────────────────────────

IMPORTS:
   import { Plus, Edit, Trash2, Eye } from 'lucide-react';
   import { toast } from 'sonner';

TOASTS:
   toast.success('✨ Sucesso!');
   toast.error('❌ Erro!');
   toast.warning('⚠️ Atenção!');
   toast.info('ℹ️ Info!');

ÍCONES:
   <Plus size={20} />        // Botões com texto
   <Edit size={18} />        // Ícones em tabelas
   <Package size={24} />     // Cards de stats

EVITAR:
   • showToast() [antigo]
   • FiArrowLeft [react-icons]
   • alert() [padrão do browser]

🔧 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────────

❌ Toast não aparece?
   ✅ Verifique se <Toaster /> está em layout.tsx

❌ Ícone não renderiza?
   ✅ Verifique import de lucide-react
   ✅ Use size={20} ou maior

❌ TypeScript error com toast?
   ✅ Instale: npm install sonner

❌ Estilo dos toasts estranho?
   ✅ Verifique CSS imports
   ✅ Verifique Tailwind config
