🔧 COMO USAR NO CÓDIGO
───────────────────────────────────────────────────────────────────────────────────

Importação:
   import { getCategoryIcon, getCategoryColor } from '@/config/categoryIcons';

Obter Ícone:
   const icon = getCategoryIcon('roupas-femininas');
   // Retorna: '👗'

Obter Cor:
   const color = getCategoryColor('acessorios');
   // Retorna: '#FFF5E5'

Configuração Completa:
   const config = getCategoryConfig('calcados', 'Calçados');
   // Retorna: { icon: '👟', color: '#F0E5FF', slug: 'calcados', label: 'Calçados' }

Lista Padrão:
   import { DEFAULT_CATEGORIES } from '@/config/categoryIcons';
   // Retorna array com 8 categorias pré-configuradas

💻 EXEMPLO PRÁTICO - Card de Categoria
───────────────────────────────────────────────────────────────────────────────────

import { getCategoryIcon, getCategoryColor } from '@/config/categoryIcons';
import Link from 'next/link';

export default function CategoryCard({ categoria }) {
  const icon = getCategoryIcon(categoria.slug, categoria.nome);
  const color = getCategoryColor(categoria.slug, categoria.nome);

  return (
    <Link href={`/produtos?categoria=${categoria.slug}`}>
      <div style={{ 
        background: color,
        padding: '20px',
        borderRadius: '8px',
        textAlign: 'center',
        display: 'flex',
        flexDirection: 'column',
        alignItems: 'center',
        gap: '10px'
      }}>
        <span style={{ fontSize: '3rem' }}>{icon}</span>
        <h3>{categoria.nome}</h3>
      </div>
    </Link>
  );
}

🎨 CUSTOMIZAÇÃO - ADICIONAR NOVA CATEGORIA
───────────────────────────────────────────────────────────────────────────────────

1. Editar: src/config/categoryIcons.ts

2. Adicionar ao CATEGORY_ICONS:
   export const CATEGORY_ICONS = {
     // ... existentes
     'jaquetas': '🧥',        // ← NOVA
   };

3. Adicionar ao CATEGORY_COLORS:
   export const CATEGORY_COLORS = {
     // ... existentes
     'jaquetas': '#E5D4FF',   // ← NOVA
   };

4. Adicionar ao DEFAULT_CATEGORIES:
   {
     id: 9,
     nome: 'Jaquetas',
     slug: 'jaquetas',
     icon: '🧥',
     color: '#E5D4FF',
   },

📊 ANTES vs DEPOIS:

ANTES:
   • Ícones duplicados em vários arquivos
   • Sem padronização de cores
   • Difícil de manter
   • Sem fallback robusto

DEPOIS:
   • Uma única fonte de verdade
   • Cores padronizadas e harmônicas
   • Fácil manutenção
   • Fallback inteligente com suporte a slug e nome

💡 ALTERNATIVAS DE ÍCONES
───────────────────────────────────────────────────────────────────────────────────

Se quiser usar IMAGENS em vez de emojis:

1. Adicione imagens em: /public/images/categories/

2. Crie mapeamento:
   export const CATEGORY_IMAGES = {
     'roupas-femininas': '/images/categories/feminino.jpg',
     'roupas-masculinas': '/images/categories/masculino.jpg',
     // ... etc
   };

3. Use no componente:
   <div style={{
     backgroundImage: `url(${CATEGORY_IMAGES[slug]})`,
     backgroundSize: 'cover',
     backgroundPosition: 'center'
   }}>
     {/* Conteúdo */}
   </div>
