📝 Guia de Uso no Admin Panel

Adicionar Cor Simples:

1. Abrir edição do produto
2. Seção "Variações" → "Cores Disponíveis"
3. Digite: Azul
4. Clique em "Adicionar"

Resultado: Bolinha AZUL SÓLIDA 🔵

Adicionar Variação com Gradiente:

1. Abrir edição do produto
2. Seção "Variações" → "Cores Disponíveis"
3. Digite: Branco/Azul
4. Clique em "Adicionar"

Resultado: Bolinha com GRADIENTE (Branco + Azul) 🟠
✅ Dicas Importantes:

✓ Use "/" sem espaços: "Branco/Azul" (não "Branco / Azul")

✓ Use exatamente os nomes da lista (maiúsculas)

✓ Cores combinam em diagonal para melhor visualização

✓ Máximo 5-7 cores por produto recomendado

✓ Tamanhos funcionam igual: P, M, G, GG

❓ Perguntas Frequentes

P: Posso usar cores que não estão na lista?

R: Sim, mas elas aparecerão como cinza. Recomendamos usar as cores da lista fornecida ou solicitar adição de novas cores ao desenvolvedor.

P: O gradiente sempre é diagonal?

R: Sim, o gradiente é aplicado a 135 graus (diagonal) para melhor visualização na bolinha. Isto é intencional.

P: Posso combinar 3 ou mais cores?

R: Atualmente suportamos 2 cores. Versão futura pode suportar 3+. Use "Cor1/Cor2" para agora.

P: Como aparecem na página de produto?

R: As cores simples aparecem como bolinhas sólidas coloridas. As variações aparecem como bolinhas com gradiente.

P: Os gradientes funcionam também no carrinho?

R: Sim! O gradiente aparece em todo lugar: listagem, produto e carrinho.
